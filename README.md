## Установка и запуск
<a id="getting_started"></a>

- Клонировать проект

```
git clone https://github.com/GaynetdinovR/fibo.git
```

- Установить зависимости

```
npm install
```

- Запустить проект

```
npm run dev
```

- Открыть в браузере

```
http://localhost:3000/
```
